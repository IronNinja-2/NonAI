Short Answer Questions

1. What is a sequential circuit?

A sequential circuit is a digital logic circuit where the output depends not only on the present inputs but also on the history of past inputs (the current state). Unlike combinational circuits, sequential circuits possess memory elements.

2. Write the names of any two basic forms of memory elements.

  • Latches
  • Flip-flops

3. What is the main difference between latches and flip-flops?

The main difference is the triggering mechanism:

  • Latch: It is level-triggered, meaning the output changes as long as the enable signal is active.
  • Flip-flop: It is edge-triggered, meaning the output changes only at a specific moment (transition) of the clock signal (rising or falling edge).

4. What will be the output (Q) of J-K flip-flop when input \(J=0\) & \(K=0\)?

The output remains unchanged (Hold State).

5. In T flip-flop, output (Q) changed from 1 to 0. What was the logic level applied at T?

The logic level was 1.
(A T flip-flop toggles its state only when the input \(T=1\)).

6. What is an asynchronous sequential circuit?

An asynchronous sequential circuit is a system that does not use a global clock signal to synchronize its internal state changes. The state transitions occur immediately when the input signals change, governed only by propagation delays.

7. What will be the output (Q) of S-R flip-flop when input \(S=1\) & \(R=1\)?

Invalid / Indeterminate. This state is forbidden because it tries to set and reset the flip-flop simultaneously, leading to unpredictable behavior.

8. How many bits can a flip-flop store?

A single flip-flop can store 1 bit of data.

9. How is a PISO register different from a SIPO register?

  • PISO (Parallel-In Serial-Out): Data is loaded into the register simultaneously (in parallel) and shifted out one bit at a time (serially).
  • SIPO (Serial-In Parallel-Out): Data enters the register one bit at a time (serially) and is read from all outputs simultaneously (in parallel).

10. Write any two applications of shift registers.

  1. Serial-to-Parallel or Parallel-to-Serial data conversion.
  2. Producing time delays in digital circuits.

11. What is meant by triggering of a Flip-flop?

Triggering refers to the specific event or signal condition that causes the flip-flop to change its state. It is usually a clock pulse transition, such as the Rising Edge (low-to-high) or Falling Edge (high-to-low).

12. Draw and explain SR latch.

An SR (Set-Reset) latch is the simplest form of bistable multivibrator. It can be constructed using two cross-coupled NOR gates (or NAND gates).

  • Set State: When \(S=1, R=0\), the output \(Q\) becomes 1.
  • Reset State: When \(S=0, R=1\), the output \(Q\) becomes 0.
  • Hold State: When \(S=0, R=0\), the latch retains its previous state.

13. Differences between JK Flip-flop and Master-Slave Flip-flop?

  • Standard JK Flip-flop: Is susceptible to the "Race Around Condition" if the clock pulse width is wider than the propagation delay when \(J=1\) and \(K=1\).
  • Master-Slave Flip-flop: Consists of two cascaded flip-flops (Master and Slave). It isolates the input from the output during the clock cycle, effectively eliminating the Race Around Condition.

14. What is a transparent latch?

A transparent latch (typically a D-Latch) is a device where the output \(Q\) follows the input \(D\) exactly as long as the Enable input is active (High). During this period, the latch is "transparent" to the data. When the Enable goes Low, the output is latched (held).

15. What is the race around condition in JK flip-flops?

The race around condition occurs when \(J=1\) and \(K=1\) (Toggle mode) and the clock pulse duration is longer than the propagation delay of the flip-flop. The output oscillates (toggles) multiple times within a single clock pulse, leading to an uncertain final state.

16. Problem with SR FF and how JK FF removes it?

Problem: The SR flip-flop has an invalid state when both inputs \(S=1\) and \(R=1\).
Solution: The JK flip-flop solves this by using feedback connections. When \(J=1\) and \(K=1\), instead of becoming invalid, the output toggles (inverts its previous state).

17. Full Adder Sum and Carry expressions.

Sum (\(S\)): \(S = A \oplus B \oplus C_{in}\)
Carry (\(C_{out}\)): \(C_{out} = AB + BC_{in} + AC_{in}\)

18. State one disadvantage of S-R Flip-flop and how to overcome it.

Disadvantage: It enters an undefined/invalid state if both inputs are High (\(S=1, R=1\)).
Overcome by: Using a JK Flip-flop, which handles the (1, 1) input condition by toggling the output.

19. State one disadvantage of J-K Flip-flops.

Disadvantage: It suffers from the Race Around Condition when \(J=1, K=1\) if the clock pulse is too wide. (This is overcome by using a Master-Slave JK flip-flop).

20. What is a synchronous counter?

A synchronous counter is a sequential circuit where all the flip-flops are triggered by the same universal clock signal simultaneously. This allows for faster operation compared to asynchronous (ripple) counters as propagation delays do not accumulate.

22. Difference between Ring and Johnson counter?

  • Ring Counter: The output of the last flip-flop is connected directly to the input of the first flip-flop. For \(n\) flip-flops, it has \(n\) distinct states.
  • Johnson Counter: The inverted output (\(\bar{Q}\)) of the last flip-flop is connected to the input of the first flip-flop. For \(n\) flip-flops, it has \(2n\) distinct states.

23. State the different types of counters.

Counters are broadly classified into:

  • Asynchronous (Ripple) Counters
  • Synchronous Counters

They can also be classified by count direction (Up, Down, Up/Down) or coding (Binary, BCD, Decade, Ring, Johnson).

24. What is the meaning of 'toggle'?

Toggle refers to the action where a flip-flop changes its output to the complement of its current state (e.g., switching from 1 to 0 or 0 to 1) upon receiving a clock pulse. This typically occurs in T flip-flops or JK flip-flops when \(J=1\) and \(K=1\).

25. List applications of counters.

  • Counting events (e.g., items on a conveyor belt).
  • Frequency division.
  • Digital clocks and timers.
  • Address generation in memory systems.

26. Differentiate between edge trigger and level trigger of a flip-flop.

  • Edge Trigger: The circuit responds to the input only at the specific transition of the clock signal (rising or falling edge).
  • Level Trigger: The circuit responds to the input as long as the clock signal is at a specific voltage level (High or Low).

Problem 3: Decade Up Counter

"Write a short note on Decade up counter."

A decade up counter (or Mod-10 counter) counts from 0 to 9 (binary 0000 to 1001) and then resets to 0. It requires 4 flip-flops. It is often implemented using a standard binary counter with a reset logic gate that clears the count when it reaches 1010 (decimal 10).

Problem 4: Ripple Counter

"What do you understand by ripple (asynchronous) counter? Draw the circuit for 3 bit ripple counter using J-K flip-flop?"

Definition:

A counter where the clock input of a flip-flop is driven by the output of the previous flip-flop rather than a global clock. This creates a "ripple" effect in delay.

Circuit Description:

It uses 3 JK flip-flops with all \(J\) and \(K\) inputs tied to Logic 1 (High). The output \(Q_0\) clocks \(FF_1\), and \(Q_1\) clocks \(FF_2\).

Problem 5: Master-Slave Flip Flop

"Illustrate the working of a Master-Slave Flip Flop discussing how it can be used to overcome the race around condition."

The Master-Slave FF typically consists of two latch stages. The "Master" is triggered on one edge of the clock (e.g., rising), and the "Slave" is triggered on the opposite edge (e.g., falling). This isolation ensures that the input cannot affect the output directly during the same clock phase, preventing the multiple toggles known as the race around condition.

Problem 6: T Flip Flop Working

"Demonstrate the working of T Flip Flop. Why is it called toggle FF?"

  • Working: When input \(T=0\), \(Q_{next} = Q\) (Hold). When \(T=1\), \(Q_{next} = \bar{Q}\) (Toggle).
  • Name: It is called "Toggle" FF because its primary function is to invert its state when the input is active.

Problem 19: JK Sequence Analysis

"Derive expression and sequence for JK Flip-flop with \(J=Q', K=1\)."

Analysis:

Expression: Standard JK characteristic equation is \(Q_{n+1} = J\bar{Q}_n + \bar{K}Q_n\).
Given \(J = \bar{Q}_n\) and \(K = 1\) (so \(\bar{K}=0\)).
Substitution: \(Q_{n+1} = (\bar{Q}_n)\bar{Q}_n + (0)Q_n = \bar{Q}_n\).

Sequence: The output simply toggles every clock pulse.
Initial: 0, Pulse 1: 1, Pulse 2: 0...

Sequence: 0, 1, 0, 1, 0, 1...

Problem 36a: Propagation Delay Calculation

"Modulo-1024 ripple counter. Frequency 1 MHz. Calculate max permissible propagation delay per FF."

Calculation:

1. Period \(T = \frac{1}{f} = 1 \mu s = 1000 \text{ ns}\).
2. Number of FFs (\(n\)) for Mod-1024: \(2^{10} = 1024\), so \(n=10\).
3. Condition: \(n \times t_{pd} \le T\)
4. \(10 \times t_{pd} \le 1000 \text{ ns} \rightarrow t_{pd} \le 100 \text{ ns}\)

Answer: 100 ns

Problem 36b: Max Frequency Calculation

"Mod-16 ripple counter. \(t_{pd} = 50\)ns. Find max frequency."

Calculation:

1. Mod-16 requires \(n=4\) flip-flops.
2. Total Delay = \(4 \times 50 \text{ ns} = 200 \text{ ns}\).
3. \(f_{max} = \frac{1}{\text{Total Delay}} = \frac{1}{200 \times 10^{-9}}\).

Answer: 5 MHz
